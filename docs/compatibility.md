# Совместимость

## Назначение
Этот документ описывает:
- какие меры могут ломать сервисы;
- какие проверки совместимости должны выполняться до применения;
- какие профили допустимы для разных типов хостов.

## Базовые категории
- сервер общего назначения;
- сервер с Docker / container runtime;
- узел Kubernetes;
- desktop / jump-host;
- чувствительные сервисы с нестандартными требованиями.
