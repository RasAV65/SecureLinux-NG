# Карта соответствия ФСТЭК

## Правила ведения
- каждый пункт ФСТЭК должен иметь статус: `not started`, `partial`, `done`, `not applicable`;
- каждый статус должен ссылаться на конкретную функцию/блок кода;
- неподтверждённые меры должны оставаться в `partial` или `not started`;
- все отклонения должны документироваться отдельно.
